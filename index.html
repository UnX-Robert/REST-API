Features

    Route /articles

  • GET request to get all the articles
  • POST request to add a new article (title, content)
  • DELETE request to delete all the articles

    • Route /articles/:articleTitle

  • GET request to get a specific article
  • PUT request to update a article (title, content)
  • PATCH request to modify a article
  • DELETE request to delete a specific article

    • Route /insert

  • GET request to insert the default items
